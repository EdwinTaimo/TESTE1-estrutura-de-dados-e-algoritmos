## Conceitos Abordados

- **Structs e typedef** — modelar entidades reais em C
- **Busca Linear** — O(n), funciona em qualquer vetor
- **Busca Binária** — O(log n), requer vetor ordenado
- **Bubble Sort** — O(n²) pior caso, O(n) melhor caso (versão otimizada)
- **Selection Sort** — O(n²) em todos os casos
- **Análise de Complexidade** — notação Big-O
